## Module Overview

This module focused on the foundations of modern frontend layout using semantic HTML, CSS and SCSS.

The work completed here moved from simple page structure and styling rules to more organized component-based layouts, responsive sections and modular SCSS files.

---

## Main Topics Covered

- Semantic HTML and content hierarchy
- Page structure with `header`, `main`, `section`, `article` and `footer`
- CSS selectors, specificity and cascade
- Box model, spacing and visual composition
- Flexbox for alignment and distribution
- CSS Grid for larger layout structure
- Responsive behavior for desktop and mobile
- Transitions and simple animations
- SCSS variables, partials and modular organization

---

## What Was Practiced

Throughout the module, the exercises and sessions were focused on building clean layouts with a realistic level of complexity for a frontend foundations course.

This included:

- writing clean semantic HTML
- separating visual assets and styles
- structuring reusable layout sections
- organizing styles into base, layout, components and sections
- building landing pages with hero areas, content rows, cards and footers
- combining Grid and Flexbox in a practical way
- keeping the project understandable for someone still learning

---

## Module Goal

The goal of this module was to build a solid base in HTML and CSS/SCSS, with attention to semantic markup, layout structure, visual organization and clean project setup.
