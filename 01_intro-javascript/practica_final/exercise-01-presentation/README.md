# Exercise 1: Developer Presentation

## Objective
Create a JavaScript object that represents your developer profile.

## Requirements
The object must include the following fields:

- `firstName` (string)
- `lastName` (string)
- `knownTopics` (array of strings) → Topics from the bootcamp you already know
- `hasProgrammingExperience` (boolean) → Indicates previous programming experience
- `socialNetworks` (array of objects)

Each object inside `socialNetworks` must contain:
- `name` (string) → e.g., "LinkedIn", "GitHub"
- `url` (string) → Link to the profile

## Technical Constraints
- Do not use external libraries.
- Use descriptive variable names in English.
- Keep the structure clean and readable.
- Add comments in the code only when necessary to clarify logic.

## Expected Outcome
A well-structured JavaScript object that accurately represents your developer profile and can be printed using `console.log()` for verification.
